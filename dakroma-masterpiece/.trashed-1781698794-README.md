# 💜🌍 DAKROMA MASTER PIECE

> Real-time multiplayer neon cyberpunk runner — playable locally in Termux and deployable online.

---

## 🚀 Quick Start (Termux / Local)

```bash
# 1. Extract ZIP
unzip dakroma-masterpiece.zip
cd dakroma-masterpiece

# 2. Install dependencies
npm install

# 3. Run server
node server.js

# 4. Open in browser
# http://localhost:3000
```

---

## 📁 Project Structure

```
dakroma-masterpiece/
├── server.js          ← Node.js server (physics, multiplayer, chat, cooldown)
├── package.json       ← Dependencies
├── names.json         ← Persistent name/cooldown store (auto-created)
├── README.md          ← This file
├── DEPLOY.md          ← Android deployment guide
└── public/
    ├── index.html     ← Full single-page app
    ├── style.css      ← Neon cyberpunk styles
    └── game.js        ← Canvas game engine + Socket.IO client
```

## 🎮 How to Play

| Action | Desktop        | Mobile     |
|--------|----------------|------------|
| Move   | Arrow Keys / WASD | ◀ ▶ buttons |
| Jump   | UP / W / Space | ⬆ button   |

**Obstacles:**
- ⚠️ **Spikes** — move in sin waves, avoid them
- ⚡ **Lasers** — flash on/off, time your dash
- 🌀 **Blades** — orbit a fixed point
- 🚀 **Jump Pads** — land on these for a huge boost!

---

## 🔐 Name Cooldown System

- Enter username → click **💾 SAVE NAME**
- Name is **locked for 20 days** on the server
- Color can be changed at any time
- Countdown timer shows remaining lock time
- All data persists in `names.json`

---

## 🌐 Multiplayer

- Up to unlimited players share the same world
- Server runs physics at 20 ticks/second
- All obstacles are seeded — everyone sees the same world
- Live leaderboard shows top 5 by score

---

## 🚀 Deployment

See **DEPLOY.md** for the full Android → GitHub → Render → Netlify guide.

**Quick summary:**
1. Push code to GitHub
2. Deploy `server.js` to **Render** (free tier)
3. Deploy `public/` to **Netlify** (free tier)
4. Update `SOCKET_URL` in `game.js` with your Render URL
